# AMAZON-Object-Area-
AIM

To compute the bounding box area of a detected object using its width and height.

PROCEDURE
Initialize width and height values
Apply the formula: Area = width × height
Substitute the given values
Compute the area
Display the result
CODE
# Input values
width = 60   # in pixels
height = 30  # in pixels

# Area calculation
area = width * height

# Output
print("Bounding Box Area:", area, "px^2")
OUTPUT

Bounding Box Area: 1800 px²

RESULT

The bounding box area is calculated as 1800 px², which represents the size of the detected object in the image.
